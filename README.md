## Installation

**Client**

```$ cd client```

```$ npm install or $ yarn install```

```$ npm start or $ yarn start```

**Server**

Change your mySQL database data ```server/index.js```

```$ cd server```

```$ npm install``` or ```$ yarn```

```$ npm rum dev```
